Class WaveformBuffer

Stores the data for a Waveform in a circular buffer, exposing methods for retrieving the data in formats that are convenient for the Display class.

The fact that a WaveformBuffer is a BuildingBlock means that it can be followed by other BuildingBlocks and IReceivers, such as filters, FileWriters, other WaveformBuffers, etc.
