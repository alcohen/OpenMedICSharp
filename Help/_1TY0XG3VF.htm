WfdbAccess.GetSignalCount

Retrieves the number of signals in the specified record. If an error occurs or the signal is not found, errMsg is filled with a meaningful error message. 

public static int GetSignalCount( string record,
	ref string errMsg );

Return Value

The number of signals if successful, otherwise 0 or negative.

Parameters

record
[path/]name of the desired record.

errMsg
If an error occurs, this contains the description of the error.
