Class FileReader

Reads the header and data from a file, typically generated by a FileWriter class, and outputs the data as a DataSource would.

Note that, for now, a FileReader will only read data from a single file; if a FileWriter wrote multiple files for a single acquisition (e.g. because they were changed every day, or were changed whenever they reached a certain size), then new functionality would be needed to read the consecutive files.
