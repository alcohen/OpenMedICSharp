TimerThread Constructor

Sets the callback function for this thread. a ThreadStart is obtained by calling: "new System.Threading.ThreadStart ( myObject.myMethod );", as in: 
...
myObj = new thisObj ();
myTimer = new System.Threading.ThreadStart ( new ThreadStart(myObj.myMethod) );
...



public  TimerThread( ThreadStart callbackFn,
	bool blocking );

Parameters

callbackFn
A valid ThreadStart (see comment for the Constructor)

blocking
if true, then the timer pauses during execution of the method,
then starts the next interval only after the method has completed.
If false, the timer starts the next interval during execution, which
means that the delay between the end of one execution and the start
of the next will be less than the specified interval, and can also
be negative (i.e. overlap of method executions).
