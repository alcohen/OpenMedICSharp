Class Sender

Specifies the Data Sender standard for OpenMedIC Building Blocks.

A Sender is a component that transmits data to a Receiver (see also IReceiver). This class, like an interface, maintains daisy-chaining compatibility in components where it's relevant, and where it's not relevant it allows to break it deliberately by not implementing Sender.

A Sender specifies an init() method to initialize the component, and two methods to add and remove Followers to the Sender. A Follower is an IReceiver.

Note that it is not unusual for a Sender to be also an IReceiver (see BuildingBlock).
