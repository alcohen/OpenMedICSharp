DataSource.getNextValue

Generates and returns the next function value to be returned, IF there is one. If there is no new value and wait = true, it waits for the next value then returns it; otherwise it leaves val unchanged and returns FALSE. 

public bool getNextValue( Sample val,
	bool wait );

Return Value

TRUE if a value was set; FALSE otherwise

Parameters

val
Sample whose value to set with the value to return

wait
If TRUE, then the function waits for data if none is
currently available; if FALSE and no data is available,
then it returns FALSE and leaves val unchanged
