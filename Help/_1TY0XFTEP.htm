Interface IReceiver

Specifies the Data Receiving interface for OpenMedIC Building Blocks.

A Receiver is a component that receives data from a Sender. The purpose for the interface is to maintain daisy-chaining compatibility in components where it's relevant, and where it's not relevant it allows to break it deliberately by not implementing IReceiver.

The interface for a Receiver specifies an init() method to initialize the component, and two methods to pass data to the Receiver: individual sample and sample array.
