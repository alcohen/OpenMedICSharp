FileWriter.maxAllowedFileSize

Maximum allowed file size, in bytes. If the next write would cause the file to exceed the specified size, then writing to that file stops. If createNewFileWhenMaxSize is TRUE, then a new file is created; this also requires that appendDateToFileName and/or appendTimeToFileName be TRUE. If createNewFileWhenMaxSize is FALSE, then all writing stops.

Note that if this value is changed after starting, then the next write will take it into account, but the previous writes will be unaffected. In other words, if you set this to less than the current file size, the size will not become smaller.
