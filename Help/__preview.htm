Class BuildingBlock

The abstract BuildingBlock class is the core specification of a building block in the OpenMedIC system.

Any OpenMedIC component that can be added to a functional sequence needs to be a BuildingBlock object. This includes any object except for Display and data sources (FileReader, Acquisition, ...?).
