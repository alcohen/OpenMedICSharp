WfdbReader Constructor

Full constructor: requires a record name and an optional path. 

public  WfdbReader( bool autoOutput,
	string path,
	string rName );

Parameters

autoOutput
If TRUE, then we push data out to the Followers AND try to
use the sample period value from the file header; if FALSE, we wait
for whoever to pull out the data.

path
A search path for the record; if null, then the default path is used.

rName
Must be an existing record name within the path.
