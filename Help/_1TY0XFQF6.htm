FileWriter.appendExistingFile

What to do if the file we are trying to create already exists, and overwriteExistingFile is FALSE. If this is TRUE, then we append to the existing file (with a new header); if this is FALSE, then we raise an exception. Note also that this cannot be TRUE if writeValuesAsAscii is false, since (at this time) binary-value files are incompatible with multiple headers.
