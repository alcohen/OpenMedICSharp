WfdbAccess.OpenAnnotation

Opens a SINGLE annotation (closing any other open annotations), according to the passed parameters, and initializing and returning a WFDB_AnninfoArray object. 

public static bool OpenAnnotation( string record,
	ref WFDB_AnninfoArray aiArray,
	string annotName,
	bool openForRead,
	ref string errMsg );

Return Value

True if successful, False if failed. If failed, the cause is in errMsg.

Parameters

record
Name of the record whose annotation we are opening.

aiArray
Annotation Info array (one element), built and populated
by this method.

annotName
Name (i.e., file extension) of the Annotation.

openForRead
True to open for reading, False to open for writing.

errMsg
Contains any error messages relevant to failed execution.
