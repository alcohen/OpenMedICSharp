WfdbAccess.GetSearchPath

Gets the current record search path for the WFDB methods. The default is typically ". /wfdb/database http://www.physionet.org/physiobank/database". Note that directory names may be mapped to DOS 8.3 format if needed. 

public static string GetSearchPath();

Return Value

The current search path, as a space-delimited string.
