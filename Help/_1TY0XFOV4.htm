DataSource.autoOutput

Determines whether data is output based on internal parameters or should wait for external input.

Internal parameters include:

-> data periodicity (e.g., a new sample is available); -> external data source has a new sample; -> local buffer (nearly?) full.

External input means a call to one of:

-> trigger(); -> getNextValue(); -> getNextValues(int).

Note that, even if autoOutput = TRUE, calls to the "external input" methods will still work, but if there is no data, getNextValue(true) will hang until the next value becomes available.
