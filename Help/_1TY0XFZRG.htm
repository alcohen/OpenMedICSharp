SquareWaveGen Constructor

Creates new instance of a square wave generator with the specified characteristics. 

public  SquareWaveGen( double secondsPerStep,
	bool autoOutput,
	double squareWaveFrequency );

Parameters

secondsPerStep
Interval between samples, in seconds or fraction thereof,
e.g. 1 KHz = 1 ms per sample would have secondsPerStep = 0.001

autoOutput
If true, then we push data out to the Followers;
if false, we wait for whoever to pull out the data

squareWaveFrequency
Frequency, in Hertz, of the generated square wave.
A value that would correspond to less than 2 samples per cycle
will throw an exception.
