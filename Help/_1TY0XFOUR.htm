DataSource.getNextValues

Generates and returns the next available function values up to vals.size. If there are less than that many values available (including 0), it returns the available values and adjusts vals.size; if there are at least that many new values available, then it returns exactly vals.size values.

The returned array has Sample[0] as the OLDEST sample; see also IReceiver.addValues(Samples). 

public void getNextValues( Samples vals );

Parameters

vals
Samples array, with .size set to the max. number of
samples desired.
