WfdbAccess.GetSignalFrame

Retrieves a frame of data for each signal in the group or record. 

public static bool GetSignalFrame( ref WFDB_SampleArray frameData,
	WFDB_SiginfoArray siArray,
	int numSignals,
	ref string errMsg );

Return Value

True if successful, False otherwise. If False, then errMsg contains the explanation why.

Parameters

frameData
Data class to be initialized and filled with data.

siArray
Signal Information array for the signals being read.

numSignals
Number of signals being read (must match the dimension of siArray).

errMsg
String where error messages, if any, are written.
