WaveformBuffer.getPoints

Retrieves the desired data points and puts them in the passed Samples object. If the requested number of points is more than are available, it returns what it has.

Note that if StartingIndex is more than MaxSeconds in the past, only the last MaxSeconds worth of data is returned. 

public long getPoints( Samples Points,
	float MaxSeconds,
	long StartingIndex );

Return Value

The actual Starting index of the returned data. This is the same as StartingIndex if StartingIndex is no more than MaxSeconds in the past.

Parameters

Points
Object for data point storage

MaxSeconds
Max. number of seconds of data to return

StartingIndex
Starting index of data to return
